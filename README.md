Bot created to add some functionalities to my personal discord server, such as checking the status of my applications, latencies and so on...

## Requirements:

- Node v21.2.0 or above.

## Setup

Make sure to install the dependencies:

```bash
# npm
npm install

# pnpm
pnpm install

# yarn
yarn install

# bun
bun install
```
